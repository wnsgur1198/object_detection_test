## Usage example

학습된 딥러닝 모델을 models폴더에 넣고, 사물인식할 이미지를 images폴더에 넣고 analyze.py를 실행시켜 지정된 이미지 내의 사물들을 검출한다.

classinfo.txt는 딥러닝 모델을 학습할 때 사용했던 데이터셋의 클래스와 일치하여야 한다.
